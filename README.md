# 🌐 Mon Portfolio

Portfolio personnel développé avec React, Vite et Tailwind CSS, permettant de présenter mon parcours, mes compétences ainsi que les différents projets réalisés durant ma formation de développeur web.

## 📖 Présentation

Ce portfolio a été conçu pour centraliser mes réalisations et mettre en avant les compétences acquises lors de ma reconversion dans le développement web.

Vous y trouverez :

- Une présentation de mon profil
- Mes compétences techniques
- Une sélection de projets réalisés
- Les liens vers les dépôts GitHub et les démonstrations en ligne
- Mes réseaux professionnels

## ⚙️ Installation

### Cloner le dépôt

```bash
git clone https://github.com/Justhiraga/Portfolio.git
```

```bash
cd Portfolio
```

### Installer les dépendances

```bash
npm install
```

---

## 🛠️ Lancer le projet

Mode développement :

```bash
npm run dev
```

L'application sera accessible à l'adresse :

```text
http://localhost:5173
```

---

## 📦 Build de production


Prévisualiser le build :
